Dog Name Generator

== Installation ==

Upload the plugin files to the `/wp-content/plugins/dng-generator` directory, or install the plugin through the WordPress plugins screen directly.

Activate the plugin through the 'Plugins' screen in WordPress.

After install, you are ready to use dng-generator in your post, page, widget or anywhere you like. How to create? lets see.

Main shortcode : [ dng-generator ]

== Initial Database ==

In `data/namesTemp.json` , there are small list of names. From this JSON file, it installs primary database named `wp_dng_data`.
